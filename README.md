# nucleate-bugs

# 🐞 App Bug Reports

Welcome!  

This repository is dedicated exclusively to reporting bugs found in Nucleate.

If you've encountered unexpected behavior, crashes, UI issues, or incorrect outputs — you're in the right place.

---

## 📌 Before Submitting a Bug

Please make sure:

- You are using the latest version of the app
- The issue is reproducible
- You have gathered any relevant logs or screenshots
- The issue is not already reported in existing Issues

---

## 📝 How to Report a Bug

1. Click **"New Issue"**
2. Select **Bug Report**
3. Fill out all required fields
4. Attach logs, screenshots, or screen recordings if possible

The more detail you provide, the faster the issue can be diagnosed and resolved.

---

## 🔎 What to Include

To help us investigate efficiently, please include:

- App version (found in Settings → About)
- Edition (Free, Pro, or Beta)
- Operating system
- Exact steps to reproduce the issue
- Expected behavior
- Actual behavior
- Error logs (if available)

---

## 📂 Attaching Logs

You can:

- Paste stack traces directly into the issue form
- Drag & drop log files into the issue
- Attach screenshots or recordings

---

## 🚫 What This Repository Is Not For

This repository is **only for confirmed bugs**.

Please do not use this space for:

- Feature requests  
- Mode submissions  
- General support questions  
- Licensing or billing issues  

Those should be directed through the appropriate channels.

---

## 🙌 Thank You

Your reports directly improve stability and reliability for everyone.

We appreciate your time and effort in helping make the app better.
